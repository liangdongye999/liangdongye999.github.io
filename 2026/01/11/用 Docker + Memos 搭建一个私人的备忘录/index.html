用 Docker + Memos 搭建一个私人的备忘录

发表于2026-01-11|更新于2026-01-11|技术

前言

为什么选择 Memos

Memos 是一个现代化的、开源的、自托管的知识管理和笔记平台，专为注重隐私的用户和组织设计。Memos 提供了一个轻量而强大的解决方案，用于捕捉、组织和分享想法，同时支持全面的 Markdown 功能和跨平台访问。

备忘录是一种原子化的、类似卡片的思维单元——旨在快速捕捉、便于浏览、并且自然组织。可以把它看作是数字化的记事卡：简短、集中、独立完整。备忘录基于这个简单的概念构建：原子化的笔记能够独立存在，同时又能连接形成知识网络。

为什么选择 Docker

Docker 是一个用于开发、交付和运行应用程序的开放平台。

Docker 使您能够将应用程序与基础设施分离，从而可以快速交付软件。

使用 Docker，您可以以管理应用程序的方式来管理基础设施。通过利用 Docker 在交付、测试和部署代码方面的方法，您可以显著减少从编写代码到在生产环境中运行代码的延迟。

使用 Docker + Cashbook 搭建一个私人的记账本

发表于2026-01-11|更新于2026-01-11|技术

前言

为什么选择 Cashbook

Cashbook 是一个功能全面、设计灵活的账单管理系统，采用前后端分离架构，后端独立管理界面便于系统维护。系统支持用户前台注册，并提供明暗主题切换以优化视觉体验。部署方式采用 Docker，简化了环境配置与发布流程。

核心功能聚焦于多平台账单导入与管理，支持支付宝、微信、京东金融的 CSV 账单文件导入，并可在导入时根据用户配置自动转换消费类型。此外，系统提供模板导入功能，满足个性化数据录入需求。

数据展示方面，系统提供直观的消费日历看板与月度账单分析（未来计划集成 AI 分析）。通过丰富的图表可视化支出类型、支付方式、每日/每月流水统计及流水归属，帮助用户清晰掌握财务状况。

系统支持多用户、多账本模式，实现用户间数据隔离与账本间独立管理。数据存储支持 PostgreSQL、MySQL 和 SQLite 多种数据库，并具备账本与系统级别的数据快速导入/导出迁移能力。附加功能包括小票图片上传、自助平账（收支抵消）以及共享账本（多用户协同记账）。同时，系统已完成 Web 移动端适配，确保在不同设备上均能流畅使用。

为什么选择 Docker

Docker 是一个用于开发、交付和运行应用程序的开放平台。

Docker 使您能够将应用程序与基础设施分离，从而可以快速交付软件。

使用 Docker，您可以以管理应用程序的方式来管理基础设施。通过利用 Docker 在交付、测试和部署代码方面的方法，您可以显著减少从编写代码到在生产环境中运行代码的延迟。

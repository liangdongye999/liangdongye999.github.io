---
title: 部署SearXNG
tags:
  - 自托管
---

用命令行：

```bash
docker run --name searxng -d \
    -p 8888:8080 \
    -v "./config/:/etc/searxng/" \
    -v "./data/:/var/cache/searxng/" \
    docker.io/searxng/searxng:latest
```

或Compose：

```yaml
versions: "3.8"

services:
  searxng:
    image: docker.io/searxng/searxng:latest
    ports:
      - 8888:8080
    volumes:
      - "./config/:/etc/searxng/"
      - "./data/:/var/cache/searxng/"  
```
